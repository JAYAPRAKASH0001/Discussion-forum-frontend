About Us

The Discussion Forum is a full-stack web application that enables users to post questions, provide answers, explore discussions, and interact through likes and dislikes. It features a clean and intuitive interface with question cards, detailed answer views, and tag-based searching for easy topic discovery. Built with Angular, Node.js (TypeScript), and MySQL, the project showcases core web development practices including CRUD operations, API integration, pagination, and dynamic UI rendering.
